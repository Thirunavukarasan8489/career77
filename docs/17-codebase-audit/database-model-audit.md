# Database Model Audit

## Identified Models
*   `User`: Standard logins credentials.
*   `Candidate`: Seeker profile properties, including `savedJobs` array.
*   `Company`: Enterprise profile fields.
*   `Job`: Job postings.
*   `Application`: Pipeline candidate applications.
*   `Interview`: Scheduled timelines.
*   `SupportTicket`: Seeker/Recruiter support desk queries.
*   `VerificationRequest`: Verification documents logs.
*   `AuditLog`: Append-only admin log entries.
