# Career77 — User Journey Design: Saved Jobs

## 1. Landing and Toggle Flow
*   **Entry Point**: Candidate browses the job boards.
*   **Action**: Clicks the bookmark button.
*   **Visual State**: Changes from outline bookmark icon to filled bookmark icon.
*   **Feedback**: A temporary toast notifies the user that the job has been saved.

## 2. Dashboard View Flow
*   **Action**: Navigates to Candidate Dashboard -> "Saved Jobs" tab.
*   **Success state**: Lists all saved jobs in a clean, high-contrast, light-themed grid layout.
*   **Empty state**: If no saved jobs exist, a friendly placeholder with a "Search Jobs" call-to-action button is displayed.
