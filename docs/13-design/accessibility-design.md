# Career77 — Accessibility Design: Saved Jobs

*   **Keyboard Navigation**: Save button is focusable via the `Tab` key.
*   **Focus Indication**: Focused items show a high-contrast ring (`focus-visible:ring-2 focus-visible:ring-indigo-600`).
*   **Screen Reader support**: Accessible properties (`aria-pressed="true/false"` and `aria-label="Save Job"`) describe toggle actions clearly.
*   **Color Contrast**: Labels and badges satisfy WCAG AA contrast rules (> 4.5:1).
