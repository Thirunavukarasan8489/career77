1. Purpose

The Admin CMS feature allows Super Admins to manage public content displayed across the Career77 platform.

Super Admins can manage:

Landing Page Content
Static Pages
FAQs
Blog Content
Announcements
Banners
Help Content
Legal Content
2. CMS Access

Route:

/admin/cms

Access requires:

Authenticated
+
Role = Super Admin
+
Account Status = Active
3. CMS Flow
Super Admin
    ↓
Admin CMS
    ↓

    ↓

    ↓

    ↓

    ↓
