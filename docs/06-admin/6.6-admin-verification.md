1. Purpose

The Admin Verification feature allows Super Admins to review and manage verification requests across the Career77 platform.

Verification may include:

Recruiter Verification
Company Verification
Company Documents
Other Platform Verification Requests

The goal is to ensure that only valid and approved users and Companies can access protected Career77 features.

2. Verification Access

Route:

/admin/verification

Access requires:

Authenticated

Role = Super Admin
+
Account Status = Active
