1. Purpose

The Subscription Management feature allows Super Admins to manage and monitor subscription plans used by Career77 users and organizations.

Super Admins can:

Create Subscription Plans
Edit Subscription Plans
Activate Plans
Deactivate Plans
View Active Subscriptions
View Subscription History
Monitor Renewals
Monitor Cancellations
Manage Subscription Status
2. Subscription Management Access

Route:

/admin/subscriptions

Access requires:

Authenticated
+
Role = Super Admin
+
Account Status = Active

34. Payment Integration

Subscription payments should integrate with the Career77 payment system.

The system should:

Verify payment status on the backend.
Validate gateway callbacks.
Prevent duplicate subscription activation.
Record successful payment events.
Handle failed payments.
Maintain transaction references.
Link payments to subscriptions.
