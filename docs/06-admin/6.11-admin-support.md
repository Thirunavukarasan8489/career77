1. Purpose

The Admin Support feature allows Super Admins to manage support requests submitted by Candidates, Recruiters, and other Career77 users.

Super Admins can:

View Support Tickets
Search Tickets
Filter Tickets
Assign Tickets
Respond to Users
Update Ticket Status
Manage Priority
Add Internal Notes
View Ticket History
Close Tickets
2. Support Access

Route:

/admin/support

Access requires:

Authenticated
+
Role = Super Admin
+
Account Status = Active

14. Ticket Conversation

The ticket should support a conversation between the User and Super Admin.

Example:

User:
I am unable to apply for this Job.

Admin:
Please share the Job ID so we can investigate.

User:
Job ID: JOB-1001

Admin:
Thank you. We are checking the issue.

The conversation should maintain message history.

15. Admin Reply

Example:
