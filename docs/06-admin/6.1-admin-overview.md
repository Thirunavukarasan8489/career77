1. Purpose

The Admin Overview defines the responsibilities, access, and main functions of the Career77 Super Admin.

The Super Admin is responsible for managing and monitoring the Career77 platform.

The Super Admin can manage:

Users
Candidates
Recruiters
Companies
Jobs
Verification
Platform Analytics
CMS
Billing
Subscriptions
Support
System Settings
2. Super Admin Access

Route:

/admin

Access requires:

Authenticated
+
Role = Super Admin
+
Account Status = Active

Only authorized Super Admin users can access the Admin Portal.

3. Super Admin Login Flow
Super Admin
    ↓
Login
    ↓
Validate Credentials
    ↓
Verify Authentication
    ↓
Check Super Admin Role
    ↓
Check Account Status
    ↓
Admin Dashboard

If authorization fails:

Access Denied

6. Recruiter Verification

Super Admin is responsible for reviewing Recruiter registration requests.

Flow:

Recruiter
    ↓
Register
    ↓
Enter Details
    ↓
Company Setup
    ↓
Submit Registration
    ↓
Super Admin Review
    │
    ├── Approve
    │      ↓
    │   Welcome Email
    │      ↓
    │   Recruiter Login
    │
    └── Reject
           ↓
       Rejection / Request Changes
           ↓
       Recruiter Can Update Details
           ↓
       Resubmit for Review

Recruiters should not access the Recruiter Portal until their account is approved.

Super Admin access should follow the principle of least privilege.

    ↓

    ↓

    ↓
Action Permission
    ↓

    ↓

Every sensitive Admin action should be authorized by the backend.
