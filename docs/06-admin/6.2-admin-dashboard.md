1. Purpose

The Admin Dashboard provides Super Admins with a centralized overview of the Career77 platform.

The Dashboard helps Super Admins monitor:

Users
Recruiters
Candidates
Companies
Jobs
Applications
Hiring
Verification Requests
Revenue
Subscriptions
Support
2. Dashboard Access

Route:

/admin/dashboard

Access requires:

Authenticated
+
Role = Super Admin
+
Account Status = Active

    ↓

    ↓

Super Admin may manually refresh Dashboard data.

Example:

Possible formats:

Excel
PDF

Exported data must respect Admin authorization and data privacy requirements.

27. Empty State

If no data is available:

No Data Available

There is not enough platform activity
to display this report.
28. Loading State

While Dashboard data is loading:

Admin Dashboard
    ↓
Loading
    ↓
Metric Card Skeletons
    ↓
Chart Skeletons
    ↓
Dashboard Loaded

Individual widgets may load independently if required.

29. Error State

If a Dashboard widget fails:

Unable to load this data.

Please try again.

Action:

[ Retry ]

A failure in one widget should not prevent other Dashboard sections from loading.

30. Data Security

The Admin Dashboard must enforce:

Authenticated User
    ↓
Super Admin Role
    ↓
Permission Check
    ↓
Dashboard Data Access

Sensitive information should only be displayed to authorized Super Admins.

Financial and personal information must be handled according to Career77's security and privacy policies.

31. Audit Logging

Dashboard-related administrative actions should be recorded when they result in data changes.

Examples:

User Suspended
Recruiter Approved
Company Verified
Job Removed
Refund Processed
Subscription Updated

Viewing Dashboard metrics does not necessarily require an Audit Log entry unless required by Career77's compliance policy.
