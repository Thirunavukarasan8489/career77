1. Purpose

The Admin Analytics feature provides Super Admins with a centralized view of Career77 platform performance and business metrics.

Super Admins can monitor:

Users
Candidates
Recruiters
Companies
Jobs
Applications
Hiring
Revenue
Subscriptions
Paid Training
Platform Activity
2. Analytics Access

Route:

/admin/analytics

Access requires:

Authenticated
+
Role = Super Admin
+
Account Status = Active

While analytics are loading:

Analytics
    ↓
Loading
    ↓

    ↓

Individual analytics sections may load independently.

35. Error State

If an analytics section fails:

Unable to load analytics data.

Please try again.

Action:

[ Retry ]

One failed analytics section should not prevent other sections from loading.

36. Data Security

The Analytics module must enforce:

Authenticated Admin
    ↓
Super Admin Role
    ↓
Permission Check
    ↓
Analytics Access

Sensitive personal information should not be exposed unnecessarily.

Analytics should preferably use aggregated data.
