# 2. URL & Route Architecture

Career77 uses role-based URL structures.

| Experience  | Login              | Register              | Application    |
| ----------- | ------------------ | --------------------- | -------------- |
| Candidate   | `/login`           | `/register`           | `/candidate/*` |
| Recruiter   | `/recruiter/login` | `/recruiter/register` | `/recruiter/*` |
| Super Admin | `/admin/login`     | Not public            | `/admin/*`     |


