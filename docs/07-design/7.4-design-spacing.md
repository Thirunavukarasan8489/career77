1. Purpose

This document defines the spacing system for the Career77 platform.

The spacing system ensures:

Consistent layouts
Clear visual hierarchy
Comfortable readability
Predictable component sizing
Responsive design
Consistent spacing across all portals

The spacing system applies to:

Public pages
Candidate portal
Recruiter portal
Super Admin portal
Authentication pages
Forms
Cards
Tables
Modals
Navigation
2. Spacing Philosophy

Career77 uses a consistent spacing scale instead of arbitrary values.

The general principle is:

Small Space
    ↓
Related Elements

Medium Space
    ↓
Content Groups

Large Space
    ↓
Major Sections

Spacing should create hierarchy without making the interface feel empty or overcrowded.

39. Spacing and Visual Hierarchy

Use spacing to communicate relationships.

Small Gap
    ↓
Strong Relationship

Medium Gap
    ↓
Related Group

Large Gap
    ↓
Separate Section

Do not use borders or backgrounds when spacing alone can communicate hierarchy.

40. Spacing Consistency

Before adding a new spacing value:

Check Existing Token
    ↓
Reuse Existing Spacing
    ↓
Add New Token Only if Required

Avoid arbitrary values such as:

13px
19px
27px
37px

unless there is a specific design requirement.

43. Future Theme Compatibility

Spacing is independent of the current Light Theme.

The same spacing system should be reused when Dark Theme is introduced.

Dark Theme development should update colors and visual surfaces without changing the fundamental spacing system unless usability testing requires it.
