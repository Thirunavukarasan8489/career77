     1. Purpose

This document defines the color system for the Career77 platform.

The current design system is based on a Light Theme only.

The Dark Theme is not part of the current implementation scope.

When Dark Theme development begins, this document must be updated with:

Dark Theme colors
Dark Theme surfaces
Dark Theme text colors
Dark Theme borders
Dark Theme status colors
Dark Theme component behavior
Theme switching rules 2. Current Theme

33. Color Contrast

All text and interactive elements should maintain sufficient contrast.

Pay special attention to:

Small text
Placeholder text
Disabled states
Status labels
Buttons
Links

The design should follow WCAG accessibility requirements.

36. Color Consistency

Before adding a new color:

Check Existing Token
↓
Reuse Existing Color
↓
Add New Token Only if Required

Do not introduce a new color for a single component without a clear design reason.
